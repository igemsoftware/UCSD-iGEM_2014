SBiDer

SBiDer contains a logically queryable database of existing genetic devices, it leverages to facilitate novel circuit design.Given a boolean set of input genes and output genes, SBiDer generates a variety of paths, via existing circuitry from the inputs to the outputs. SBiDer is iGEM Gold Medal winning web application, which continuously evolves with the help of fellow scientists.
